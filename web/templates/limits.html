Limits

Last updated: May 24, 2026

ShareIt offers two tiers of service depending on whether you are logged in or accessing the service as a guest. Below are the current limits for each tier.

Guest (Anonymous)

  • Maximum file size: 750 MB
  • Retention period: 7 days
  • Rate Limit: Dynamic, based on Server Load and Usage Patterns

Authenticated (CNS Account)

  • Maximum file size: 1.5 GB
  • Retention period: 90 days
  • Rate Limit: Dynamic, based on Server Load and Usage Patterns. Higher allowance window
  • Device management: File sync between devices
  • Recent uploads: Browse and re-download your file history from any device

Rate Limits

To ensure fair usage and prevent abuse, the following rate limits are enforced per IP address and Session:

  • Standard: No requests limited
  • Limited: ~5 uploads per minute
  • Restricted: No requests allowed (Special Cases or Violations)

These limits are deliberately conservative during development and can be adjusted dynamically and automatically.

File Expiration

All uploaded files are automatically deleted from our servers after their retention period expires. We do not retain copies of expired files. The expiration timer starts at the moment the file upload is finalized.

Contact

If you need higher limits or have questions about these restrictions, please contact us at contact@cns-studios.com.
