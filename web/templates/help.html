Help

Last updated: May 24, 2026

How do I share a file?

You have two options to share a file. You can either create a direct Quick Share to quickly transfer many files between devices and people, or you can create a Link, which is more suitable for sharing a file with many people for a longer time.

How long are files stored?

Guest (non-logged-in users) uploads are stored for 7 days. Logged-In users get 90 days of storage. After the retention period expires, the file is automatically and permanently deleted. More about that in our Limits Page.

What is the maximum file size I can upload?

Guest (non-logged-in users) can upload files up to 750 MB. Logged-In users can upload files up to 1.5 GB. More about that in our Limits Page.

Is my file secure?

Yes. All files are encrypted with AES-256-GCM before they are stored. The link you share contains the key needed for decryption. TLS 1.3 protects the data during transit. We cannot view any file contents by design.

Can I cancel an upload?

No, once an upload has started, it cannot be canceled. However, we cannot view any files, so if you accidentally upload a file, you can simply not share the link that has been copied to your clipboard. The file will be automatically deleted after the retention period.

What happens if the link is wrong or broken?

If you visit a share link that shows a "Wrong Link" message, double-check the URL you received and make sure you copied the entire link. If the Link still doesn't work, ask the sender to send the link again, or ask the sender to upload the file again

How much can I upload?

You can upload as many files as you want, as long as the files stay in the limits of your account type (Guest or Logged-In). If you got "Rate Limited", just wait a bit and try again. This is to ensure our servers remain accessible for everyone.
