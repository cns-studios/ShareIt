Help

Last updated: April 5, 2026

How do I share a file?

Go to the homepage, upload your file, and you will receive a shareable link. Send this link to anyone you want to share the file with. The link contains the decryption information, so only people with the link can access the file.

How long are files stored?

Guest uploads are stored for 7 days. Authenticated users get 90 days of storage. After the retention period expires, the file is automatically and permanently deleted.

What is the maximum file size I can upload?

Guest users can upload files up to 750 MB. Authenticated users can upload files up to 1.5 GB.

Is my file secure?

Yes. All files are encrypted with AES-256-GCM before they are stored. The link you share contains the key needed for decryption. TLS 1.3 protects the data during transit.

How do I download a file I received?

Click the link the sender shared with you. The page will automatically load the file details. If no password is required, the download will start automatically or present a download button depending on the settings chosen by the sender.

Can I cancel an upload?

Yes, during the upload process you can cancel by clicking the cancel button. This will abort the upload and clean up any temporary data.

What happens if the link is wrong or broken?

If you visit a share link that does not contain a valid password or session, the page will show a "Wrong Link" message. Double-check the URL you received and make sure you copied the entire link.

How do I report an issue?

If you encounter a problem, please use the "Report abuse" link on the download page or contact us at contact@cns-studios.com.
